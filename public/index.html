Telegram Benachrichtigungen

Lass dich sofort per Telegram informieren, wenn ein überwachter Server offline geht.

Willkommen beim HomeServer Hub! Hier findest du eine einfache Schritt-für-Schritt-Anleitung, wie du deine Geräte verwaltest und Benachrichtigungen einrichtest.

1. Erste Schritte

Das Dashboard prüft alle 30 Sekunden automatisch den Status deiner Geräte. Drücke auf „Jetzt prüfen“ im Header, um einen manuellen Check auszulösen.

2. Geräte suchen (Netzwerk-Scan)

Du musst Adressen nicht manuell eintippen! Klicke oben auf „Netzwerk scannen“ und dann auf „Scan starten“.

Wähle nach dem Scan die gefundenen Dienste (wie Home Assistant, Plex oder Docker-Container) aus und importiere sie mit einem Klick.

3. Telegram-Ausfallwarnung einrichten

Du kannst dich per Telegram informieren lassen, wenn ein Server offline geht. So erstellst du den dafür nötigen Bot:

A. Bot erstellen (Bot-Token erhalten):
  1. Suche in Telegram nach dem Kontakt @BotFather (offizielles Profil mit blauem Haken).
  2. Sende ihm den Befehl /newbot.
  3. Gib einen Anzeigenamen ein (z. B. Mein Server Wächter).
  4. Gib einen Benutzernamen ein, der auf bot endet (z. B. mein_waechter_hub_bot).
  5. Kopiere das angezeigte Token (z. B. 123456789:ABCdefGh...).
B. Deine Chat-ID ermitteln:
  1. Suche in Telegram nach dem Kontakt @userinfobot.
  2. Klicke auf Start. Er sendet dir sofort deine ID-Nummer zurück (z. B. 987654321).
C. Im Dashboard speichern:

Wähle oben den Tab „Telegram“, trage Bot-Token und Chat-ID ein, klicke auf „Verbindung testen“ und speichere danach die Änderungen.

4. Favoriten & Einklappen

Favoriten: Klicke auf den Stern oben links einer Kachel. Sie wird dann ganz oben im Schnellzugriff-Bereich angeheftet.

Einklappen: Klicke auf den Titel einer Kategorie (z. B. „Media“), um sie einzuklappen. Der Zustand wird automatisch gespeichert.
